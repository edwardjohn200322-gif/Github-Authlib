# GitHub OAuth 2.0 Flask Application

A simple Flask web application that demonstrates GitHub OAuth 2.0 authentication using Authlib. The application allows users to log in with their GitHub account, display their GitHub profile information, and protect routes using session-based authentication.

---

## Features

- GitHub OAuth 2.0 Login
- Protected Profile Route
- Session-based Authentication
- Logout Functionality
- Display GitHub User Information
- Flask + Authlib Integration

---

## Technologies Used

- Python
- Flask
- Authlib
- GitHub OAuth API

---

## Installation

pip install flask authlib requests

Running the Application

Start the Flask server:

python app.py

Open browser:

http://localhost:5000
